# Estrategia de testing de skills

Checklist recomendado:

1. Trigger test: la descripcion del frontmatter activa la skill correcta.
2. Conflict test: cuando hay varias skills parecidas, gana la mas especifica.
3. Regression test: cambios en una skill no rompen workflows existentes.
4. Prompt quality: instrucciones claras, cortas y verificables.
