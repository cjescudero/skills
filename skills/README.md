# Skills

Cada skill debe vivir en su propia carpeta:

`skills/<nombre>/SKILL.md`

Recomendaciones:
- usa frontmatter con `name` y `description`,
- en `description`, describe claramente cuando debe activarse,
- evita reglas ambiguas o demasiado generales.

- `bus-arrivals-coruna-data`
